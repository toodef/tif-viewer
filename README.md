## Supported file formats:
- [x] *.tif
  - [x] float32
  - [ ] float16
  - [ ] uint8
- [x] *.tiff
  - [x] float32
  - [ ] float16
  - [ ] uint8
- [ ] *.exr

## Features:
- [x] Fast threshold
- [x] Image info
- [ ] Comparision mode
- [ ] Big image partial uploading
- [ ] Color mapping

## System requirements:
- 1 core CPU or more
- OpenGL v3.0 or highter
- 4 Gb RAM or more
